Features

Everything you need.
Nothing you don't.

One Go binary. One PostgreSQL database. Every feature built for agent consumption.

Bank Sync

Connect via Plaid, Teller, or CSV import. Transactions and balances sync automatically on a schedule.

MCP Server

30+ tools for querying, categorizing, and managing transactions. Streamable HTTP or stdio transport.

Transaction Rules

Recursive AND/OR/NOT conditions. Agents create rules that auto-categorize future transactions during sync.

Review Queue

New and uncategorized transactions surface for triage. Agents or humans approve, reject, or skip.

REST API

Full CRUD with cursor pagination, field selection, and composable filters. Build your own dashboards.

Privacy First

Self-hosted. AES-256-GCM encryption at rest. Your data never leaves your server. Export or delete anytime.

Quick Start

Running in 60 seconds

Docker Compose handles everything. PostgreSQL, migrations, and the server.

# Clone and configure
$ git clone https://github.com/canalesb93/breadbox.git
$ cd breadbox && cp .env.example .docker.env
$ echo "ENCRYPTION_KEY=$(openssl rand -hex 32)" >> .docker.env

# Launch
$ docker compose up -d

# Visit http://localhost:8080 to start the setup wizard
