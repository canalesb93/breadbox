API Key OAuth
  1. Open Claude Desktop and go to Settings
  2. Navigate to the Developer tab
  3. Click Edit Config to open claude_desktop_config.json
  4. Add the Breadbox MCP server entry (switch to Config tab to see the snippet)
  5. Save the file and restart Claude Desktop
  6. Breadbox tools should appear in Claude's tool list
Claude Desktop supports OAuth natively. If you omit the API key headers, Claude will automatically initiate the OAuth consent flow when connecting.

Config file location:

macOS: ~/Library/Application Support/Claude/claude_desktop_config.json

Windows: %APPDATA%\Claude\claude_desktop_config.json
