How bulk edit works

Export loads the current data into the text area as tab-separated values. Edit the text (or paste from a spreadsheet), then click Import. By default, existing entries are updated and new ones are created — rows you remove are not deleted. Enable Full replace to also delete any entries not present in the import.

Merging categories: To consolidate categories, set the merge_into column to the target category's slug. All transactions and mappings from the source category are moved to the target, then the source is deleted. Great for simplifying a complex taxonomy. Use the button to copy instructions for an AI agent to help you edit the file.
