Teller Configuration

Teller requires mTLS certificates that must be configured via environment variables. Set the following before starting Breadbox:

TELLER_APP_ID=your-app-id
TELLER_CERT_PATH=/path/to/certificate.pem
TELLER_KEY_PATH=/path/to/private_key.pem
TELLER_ENV=sandbox

For Docker Compose, add these to your .env file and mount the certificate files as volumes.
